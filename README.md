# Simple Network Sniffer

A beginner-friendly network sniffer built using Python and Scapy.

## Features
- Captures network packets
- Displays source and destination IP addresses
- Detects TCP and UDP protocols
- Shows source and destination ports
- Displays packet timestamps

## Technologies Used
- Python
- Scapy

## How to Run

```bash
pip install scapy
python network_sniffer.py
```
